Space Journey 🚀
A collection of artificial intelligence and machine learning projects focused on problems in astronomy, astrophysics, and aerospace engineering.

This repository documents a learning and research journey through space-related data science problems using modern machine learning techniques.

Projects
Exoplanet Detection (Kepler Data)
Machine learning models are used to classify potential exoplanets using the NASA Kepler Objects of Interest (KOI) dataset.

Key techniques:

  Data exploration and visualization
  Feature engineering from astrophysical parameters
  Classical machine learning models
  Model evaluation and comparison
  More projects will be added as the journey continues.

Goals:
  Apply machine learning to real scientific datasets
  Build strong AI projects for research and engineering
  Explore intersections between AI, astronomy, and aerospace
  
Technologies:
  Python
  Scikit-learn
  XGBoost / LightGBM
  Pandas
  NumPy
  Matplotlib / Seaborn
  Jupyter Notebook
